# vue-ruby-china

> Vue 实现的RubyChina社区

## 技术栈
* vue
* vue-router
* vuex
* vue-resource
* bootstrap
* ES6

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
